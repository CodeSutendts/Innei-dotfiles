# Innei's Dotfiles

### Usage

Install rcm first. then run 

```bash
cd
git clone https://github.com/Innei/dotfiles --depth=1 ~/.dotfiles
rcup
```

After, you will got normal config. (Eg. Tmux, zsh, git ...)

### Set up other config

nvim, ranger config can set up with `-t`

```sh
rcup -t nvim
rcup -t ranger
```
